# JDBC Interview Questions (0–3 YOE)

---

## 0 Level

1. What is JDBC?
2. What problem does JDBC solve?
3. What is JDBC driver?

---

## Fresher Level

1. Explain JDBC workflow.
2. What is Connection?
3. Difference between executeQuery and executeUpdate?
4. What is ResultSet?
5. Why must resources be closed?

## 1–2 YOE

1. Difference between Statement and PreparedStatement?
2. What is SQL Injection?
3. What is auto-commit?
4. How do you manage transactions in JDBC?
5. Why is PreparedStatement faster?

## 2–3 YOE

1. What happens if connection is not closed?
2. Why is connection creation expensive?
3. How does JDBC ensure database independence?
4. How would you handle failure between two DB operations?
5. Why is try-with-resources preferred?
